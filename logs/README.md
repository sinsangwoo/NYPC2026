# Logs 폴더

게임 로그와 분석 결과를 저장하는 폴더입니다.

## 폴더 구조

```
logs/
├── raw/       # 원본 게임 로그 저장 폴더
└── analysis/  # 분석 결과 저장 폴더
```

## 각 폴더/파일별 역할

### `raw/` 폴더
- **목적**: 자동 경기 실행 스크립트(`scripts/run_matches.py`)로 생성된 원본 게임 로그를 저장합니다.
- **파일 형식**: `game_0001.log`, `game_0002.log`, ..., `game_0100.log`
- **사용 방법**: `scripts/run_matches.py`를 실행하면 자동으로 이 폴더에 로그가 저장됩니다.

### `analysis/` 폴더
- **목적**: 게임 로그를 분석한 결과를 저장합니다.
- **사용 방법**: 분석 스크립트를 실행하면 이 폴더에 결과가 저장됩니다.
